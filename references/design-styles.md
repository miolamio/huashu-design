## 1. Information Architecture School (01-04)
> Philosophy: "Data is not decoration; it is building material."

### 01. Pentagram - Michael Bierut Style
**Philosophy**: type is language; grid is thought.
**Key Traits**:
- Extremely restrained color: black and white plus one brand color
- A modern interpretation of the Swiss grid system
- Typography as the primary visual language
- Strategic use of negative space, with 60%+ whitespace

**Prompt DNA**:
```
Pentagram/Michael Bierut style:
- Extreme typographic hierarchy, Helvetica/Univers family
- Swiss grid with precise mathematical spacing
- Black/white + one accent color (#HEX)
- Information architecture as visual structure
- 60%+ whitespace ratio
- Data visualization as primary decoration
```

**Representative Work**: Hillary Clinton 2016 campaign identity
**Search Keywords**: pentagram hillary logo system

## Prompt Usage Guide

**Combination formula**: `[style prompt DNA] + [scene template (see scene-templates.md)] + [specific content]`

### Core Principle: Mood, Not Layout

The key to AI image generation: short prompts outperform long prompts. Three sentences describing mood and content usually work better than 30 lines of layout detail.

| Diversity-killing phrasing | Creativity-triggering phrasing |
|----------------|----------------|
| Specify color ratios (60%/25%/15%) | Describe the mood ("warm like Sunday morning") |
| Mandate layout positions ("title centered, image on the right") | Reference specific aesthetics ("Pentagram editorial feel") |
| Restrict character poses and expressions | Let the AI interpret the style naturally |
| List every visual element to include | Describe what the audience should feel |

### Good / Bad Examples

**Bad — over-constrained, producing empty and flat AI output:**
```
Professional presentation slide. Dark background, light text.
Title centered at top. Two columns below. Left column: bullet points.
Right column: bar chart. Colors: navy 60%, white 30%, gold 10%.
Font size: title 36pt, body 18pt. Margins: 40px all sides.
```

**Good — mood-driven, producing more varied and tactile results:**
```
A data visualization that feels like a Bloomberg Businessweek
editorial spread. The key number "28.5%" should dominate the
composition like a headline. Warm cream tones with sharp black
typography. The data tells a story of dramatic channel shift.
```

### Choosing an Execution Path

Choose based on the "Best Path" column in the quick reference table:
- **AI Generation**: styles with explicit visual elements, such as 06/07/12/13/14/15/16/20, can be produced directly with Gemini/Midjourney
- **HTML Rendering**: styles that depend on precise typography, such as 01/03/04/10/11/17/18, should use code to control data and layout
- **Hybrid**: HTML for the structural layout plus AI-generated imagery or backgrounds, such as 02/05/08/09/19

### Quality Control

1. ❌ Do not write "in the style of Pentagram" directly → ✅ Describe concrete design traits
2. Text often fails in AI generation → replace text after generation
3. Proportions distort easily → specify aspect ratio clearly
4. Generate 3-5 variants first, then refine the best one

**Default aesthetic no-go zones** (users may override based on their brand):
- ❌ Cyber neon / dark-blue background (#0D1117)
- ❌ Cover images with personal signatures or watermarks
