## Usage Guidelines

### Volume Recommendations (from the audio dual-track system in apple-gallery-showcase.md)
- **SFX main track**: `1.0` (no attenuation)
- **BGM background track**: `0.4 ~ 0.5` (SFX clearly cuts through)
- **Multiple SFX layered**: use `amix=inputs=N:duration=longest:normalize=0` to preserve dynamic range

### ffmpeg Stitching Templates
```bash
# Align a single SFX to a timestamp:
ffmpeg -i video.mp4 -itsoffset 2.5 -i sfx/ui/click.mp3 \
  -filter_complex "[0:a][1:a]amix=inputs=2:duration=longest:normalize=0[a]" \
  -map 0:v -map "[a]" output.mp4

# Multiple SFX + BGM:
ffmpeg -i video.mp4 \
  -itsoffset 1.0 -i sfx/transition/whoosh-fast.mp3 \
  -itsoffset 1.6 -i sfx/impact/logo-reveal-v2.mp3 \
  -i bgm.mp3 \
  -filter_complex "[3:a]volume=0.4[bgm];[0:a][1:a][2:a][bgm]amix=inputs=4:normalize=0[a]" \
  -map 0:v -map "[a]" output.mp4
```

### Selection Decision Tree
1. **Tactile action** (typing/clicking/swiping) → `keyboard/` or `ui/`
2. **Element enters/exits** → `transition/`
3. **Container-level operation** (card/modal) → `container/`
4. **State feedback** (success/failure/notification) → `feedback/`
5. **Progress/time passing** → `progress/`
6. **Brand landing point/important moment** → `impact/`
7. **AI magic/transformation** → `magic/`
8. **Terminal/code demo** → `terminal/`

### Avoid Audio Pileups
- Allow a maximum of `2 SFX` concurrently at the same timestamp
- You can use 3 when BGM is below 0.3
- During a brand impact, clear other SFX (leave 0.2s of space before the landing point)

## Prompt Writing Principles (For Reuse)

Reference style: `apple keynote, tight, minimal, no reverb unless ambient, crisp, elegant`

**The three elements of a good prompt**:
1. **Physical description of the sound**: what object, what action ("mechanical keyboard single key press")
2. **Texture/style constraints**: apple-style / ios-style / cinematic / retro
3. **Negative constraints**: no reverb / clean studio / minimal

❌ "click sound"
✅ "crisp ui button click, clean modern interface sound, apple-style, high pitched"

❌ "magic"
✅ "bright twinkling stars sound, high pitched glittery chime, fairy dust"
